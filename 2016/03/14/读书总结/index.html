  1. 初始化
    void InitStack(&S){
    s.top=-1;//将栈顶指针置为空

    bool push(x){
    if (S.top==MaxSize-1) return false;//栈满，报错

  4. 出栈
    bool pop(x){
    if (S.top==-1) return false;//栈空，报错
    x=S.data[S.top–];//先出栈，指针再减1
    return true;
    }
  5. 读栈顶元素

  2. 判队空
    bool isEmpty(Q){

  3. 入队
    bool EnQueue(Queue &Q,x){
    if ((Q.rear+1)%MaxSize==Q.front) return false;//队满
    Q.data[Q.rear]=x;
    Q.rear=(Q.rear+1)%MaxSize;//队尾指针加1取模
    return true;
    }
  4. 出队
    bool DeQueue(Queue &Q,&x){
    if (Q.rear==Q.front)return false;
    x=Q.data[Q.front];
    Q.front=(Q.front+1)%MaxSize;
    return true;
    }

插入排序

直接插入排序

思想：表长L[1…..N],有序序列L[1..i-1],L(i),无序序列L[n+1……N],将L(i)插入有序L[1..i-1]。 

- 找出L(i)插入有序L[1..i-1]的位置k。  
- 将L[k..i-1]中所有元素后移一位  
- 将L(i)复制给L(k)。

空间效率：O（1）
时间效率：O（N2）

冒泡排序

思想：表长n，从前往后或（从后往前）两两比较相邻元素的值，逆序则交换，这是第一趟，需要n-1趟。
空间效率：O（1）
时间效率：平均和最坏都是：O（N2）

快速排序

是所有排序算法中平均时间最好的一种算法，O(nlogn)思想是基于分治法。

  1. 调整序列中的元素，使得当前序列最左短的元素在调整后满足左侧所以元素均不超过该元素、右侧所有元素均大于该元素。
  2. 对该元素的左侧和右侧分别递归进行1的调整，直到当前调整区间的长度不超过1.
    时间复杂度：O(n2)
    空间复杂度：O(logn)

two pointers

  1. 题目：给定一个递增的正整数序列和一个正整数M,求序列中的两个不同位置的数a和b,使他们的和恰好为M，输出所有满足条件的方案。
    思路：令下标i的初值为0，下标j的初值为n-1，令i，j分别指向序列的第一个元素和最后一个元素，接下来根据a[i]+a[j]与M的大小来进行下面三种选择，i向右移动，j向左移动，直到i>=j成立。
    1>如果a[i]+a[j]==M，令i=i+1，j=j-1;
    2>如果a[i]+a[j]>M,j=j-1;
    3>如果a[i]+a[j]<M,i=i+1;
    代码：时间复杂度O(n)

  2. 题目：假设有两个递增序列A和B，要求将它们合并为一个递增序列C。
