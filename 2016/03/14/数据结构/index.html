    简单计算器（ACM）

    读入一个只包含 +, -, *, / 的非负整数计算表达式，计算该表达式的值。

输入
测试输入包含若干测试用例，每个测试用例占一行，每行不超过200个字符，整数和运算符之间用一个空格分隔。没有非法表达式。当一行中只有0时输入结束，相应的结果不要输出。

输出
对每个测试用例输出1行，即该表达式的值，精确到小数点后2位。

样例输入
30 / 90 - 26 + 97 - 5 - 6 - 13 / 88 6 + 51 / 29 + 79 87 + 57 * 92
0
样例输出
12178.21
思路：题目给出的是中缀表达式

- 中缀表达式转后缀表达式 
 1>设立一个操作符栈，存放操作符，设立一个队列，存放后缀表达式
 2>扫描中缀表达式，遇到操作数放入后缀表达式。
 3>遇到操作符op，则将其优先级与操作符栈顶操作符优先级比较; 
 若高于，则压入栈，若低于，则弹出到后缀表达式。
 4>重复直到扫描完，若仍有，放到后缀表达式
- 计算后缀表达式
扫描后缀表达式，若是操作数，就压入栈，若是操作符，就弹出两个操作数，进行计算，计算后的数压入栈。直到扫描完，栈中只有一个数。

答案：http://blog.csdn.net/ccj2020/article/details/7749450

    void Initqueue(&Q){
    Q.rear=Q.front=0;
    }
  2. 判队空
    bool isEmpty(Q){
    if (Q.rear==Q.front) return true;
    else return false;
    }
  3. 入队
    bool EnQueue(Queue &Q,x){
    if ((Q.rear+1)%MaxSize==Q.front) return false;//队满
    Q.data[Q.rear]=x;
    Q.rear=(Q.rear+1)%MaxSize;//队尾指针加1取模
    return true;
    }
  4. 出队
    bool DeQueue(Queue &Q,&x){
    if (Q.rear==Q.front)return false;
    x=Q.data[Q.front];
    Q.front=(Q.front+1)%MaxSize;
    return true;
    }

排序（内排）

插入排序

直接插入排序

思想：表长L[1…..N],有序序列L[1..i-1],L(i),无序序列L[n+1……N],将L(i)插入有序L[1..i-1]。 

- 找出L(i)插入有序L[1..i-1]的位置k。  
- 将L[k..i-1]中所有元素后移一位  
- 将L(i)复制给L(k)。

空间效率：O（1）
时间效率：O（N2）

冒泡排序

思想：表长n，从前往后或（从后往前）两两比较相邻元素的值，逆序则交换，这是第一趟，需要n-1趟。
空间效率：O（1）
时间效率：平均和最坏都是：O（N2）

快速排序

是所有排序算法中平均时间最好的一种算法，O(nlogn)思想是基于分治法。

  1. 调整序列中的元素，使得当前序列最左短的元素在调整后满足左侧所以元素均不超过该元素、右侧所有元素均大于该元素。
  2. 对该元素的左侧和右侧分别递归进行1的调整，直到当前调整区间的长度不超过1.
    时间复杂度：O(n2)
    空间复杂度：O(logn)

简单选择排序

思想：表长N[1…..N],第i趟排序则从[i……N]中选择关键字最小的元素与i交换，每一趟确定一个元素的最终位置，需要n-1趟。
